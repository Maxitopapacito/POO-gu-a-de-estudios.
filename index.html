01 Los 5 Patrones del Profe

Patrón 1 — Clase Simple e Inmutable SIMPLE

Todas las propiedades son de solo lectura (get sin set). Todo valor se asigna únicamente en el constructor.

Si alguien puede escribir consola.Id = 99 desde fuera, rompes toda la lógica del programa.
public class Consola
{
    public int    Id          { get; }   // sin set = solo lectura
    public string Descripcion { get; }

    public Consola(int id, string descripcion)
    {
        Id = id;
        Descripcion = descripcion;
    }
}

Mismo patrón: Day, Time, Genre, Patient, Renta, Videojuego…

Patrón 1b — Propiedad Calculada

Definida con =>. No guarda valor en memoria — lo calcula al vuelo cada vez que se accede.

public string FullName => $"{LastName}, {FirstName}";
// No hay campo, no hay set. Se recalcula en cada acceso.

internal List<Videojuego> VideojuegosDisponibles(int consolaId, int folio)
{
    List<Videojuego> resultados = new List<Videojuego>();
    List<RentaVideojuego> rentados =
        rentaVideojuegos.FindAll(r => r.Folio == folio);

    foreach (var vj in videojuegos)
    {
        if (vj.ConsolaId != consolaId)                       continue; // filtro 1
        if (rentados.Exists(r => r.VideojuegoId == vj.Id)) continue; // filtro 2
        resultados.Add(vj);
    }
    return resultados;
}

03 Errores Comunes

Error 1 — Variable local que tapa al campo de clase

❌ Mal
internal List<Consola> Disponibles()
{
    // VACÍA — tapa this.consolas
    List<Consola> consolas =
        new List<Consola>();
    foreach (var c in consolas) // itera vacía
    { ... }
}
✓ Bien
internal List<Consola> Disponibles()
{
    // Copia de this.consolas
    return new List<Consola>(consolas);
}

Error 2 — while vacío

❌ Mal
switch (option) { ... }
while (option != 0); // ← el ; lo termina
✓ Bien
do
{
    // menú completo aquí
    switch (option) { ... }
} while (option != 0);

Error 3 — static dentro de Main

❌ Mal — no compila
static void Main(string[] args)
{
    static VideoClub club = new VideoClub();
}
✓ Bien — nivel de clase
internal class Program
{
    static VideoClub videoclub = new VideoClub();
    static void Main(string[] args) { ... }
}

Error 4 — List con paréntesis en vez de llaves

❌ Mal
rentas = new List<Renta>(
    new Renta(001, "Juan") // error
);
✓ Bien
rentas = new List<Renta>()
{
    new Renta(001, "Juan"),
    new Renta(002, "Valentina")
};
